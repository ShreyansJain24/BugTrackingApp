Bug Tracking Application

Project Features

  • Application Description:-

    App allows the user to report the bug so that it could further be assigned/tracked along with specifying its priority and description to replicate the bug in current systems.

  • Technical Details:-

    Front end for the application has been designed in Angular and Backend has been developed in spring boot (java),Hibernate and MySql.

  • Features:-

    • List all reported bugs with details such as BugId,title, description, priority,etc.
    • Implement pagination, sorting, and filtering options for better user experience. using mat paginator.
    • Provide a form to add a new bug with fields for title, description, priority, and any other relevant information
    • Enable users to modify bug details, such as updating the title, description, priority, status, etc
    • Enabled user to perform CRUD(Create Read,Update Delete) operation for Bugs Raising
